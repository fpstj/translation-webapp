## `docs/TROUBLESHOOTING.md`

# Troubleshooting

## Common Issues

### App doesn't start

- Check Node version
- Run `npm install` again
- Check `.env` setup

### API errors

- Is the backend running?
- Check `VITE_API_URL` value

### Styles not applying

- Confirm Tailwind is configured correctly
